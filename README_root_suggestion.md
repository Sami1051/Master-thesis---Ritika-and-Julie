# Master-thesis---Ritika-and-Julie

Techno-economic model for a case study of mini-scale CCS at Trehørningen Energisentral
(Eidsiva Bioenergi AS, Hamar, Norway).

## Structure

- `model/CCS_TEA_Model.ipynb` — the authoritative model (deterministic TEA baseline, 10,000-iteration
  Monte Carlo, structural scenario analysis) used for all results reported in the thesis.
- `data/processed/` — cleaned input data (EU ETS Phase 4 auction prices, NO1 electricity prices).
- `outputs/` — figures generated by the model.
- `appendix/` — derivations and validation tests referenced in the thesis (see `appendix/README.md`
  for what each file fulfills).
- `requirements.txt` — pinned package versions.

## Reproducing the submitted results

The exact commit used to generate the thesis's reported results is tagged `thesis-submission-v1`:

```
git checkout thesis-submission-v1
pip install -r requirements.txt
```

Then run `model/CCS_TEA_Model.ipynb` top to bottom (fixed random seeds throughout, per §4.2.2 of the
thesis, so results reproduce exactly).
